# Frans Bilas — Qualidade de Software 1

Bem-vindo a documentacao do projeto da equipe **Frans Bilas** para a disciplina FGA315 — Qualidade de Software 1 (2026.1), ministrada pela Profa. Cristiane Ramos na FCTE/UnB.

## Sobre o Projeto

Este projeto consiste na avaliacao da qualidade de produto do software **Agio** — uma Aplicacao de Gestao de Inventario Otimizada, desenvolvida como projeto da disciplina MDS (UnB, 2024.2). O Agio e um sistema web open source que permite a empresas de pequeno e medio porte gerenciar seus inventarios de forma pratica e segura.

- **Repositorio do Agio**: [github.com/unb-mds/2024-2-Agio](https://github.com/unb-mds/2024-2-Agio)
- **Deploy**: [agio-inventory-system.vercel.app](https://agio-inventory-system.vercel.app/)

## Caracteristicas de Qualidade Avaliadas

A avaliacao segue a norma ISO/IEC 25010 (SQuaRE), com foco em tres caracteristicas:

| Caracteristica | Subcaracteristicas | Por que avaliar? |
|---|---|---|
| Adequacao Funcional | Completude, Correcao, Adequacao | Verificar se o que foi planejado no backlog foi implementado corretamente |
| Confiabilidade | Maturidade, Disponibilidade, Tolerancia a Falhas | Avaliar se o sistema e estavel para uso em producao |
| Eficiencia de Desempenho | Comportamento Temporal, Utilizacao de Recursos, Capacidade | Medir se o desempenho atende ao perfil de uso esperado |
